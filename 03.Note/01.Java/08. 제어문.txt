[Java 조건 및 If 문]
Java는 수학의 일반적인 논리 조건사용

미만: < b
작거나 같음: a <= b
보다 큼: > b
크거나 같음: >= b
a == b와 같음
같지 않음: a != b

1. if문

if (조건문) {
  // 실행코드
} else if (조건문) {
  // 실행코드
} else {
  // 실행코드
}

if지정된 조건이 true인 경우 실행할 코드
else동일한 조건이 false인 경우 실행할 코드
else if첫 번째 조건이 false인 경우 테스트할 새 조건을 지정하는 데 사용

2. 삼항연산자(짧은 if문)
변수 = (조건문) ? true시 실행문 :  false시 실행문;


3. switch 문

: 단일조건을 분류하여 실행코드를 나눔
switch(변수) {
  case x:
    // 실행코드
    break;
  case y:
    // 실행코드
    break;
  default:
    // 실행코드
}
